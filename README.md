# Rank Dichotomy Theorem (CAT(0) Cube Complexes)
DOI: https://doi.org/10.5281/zenodo.18450375

Status: Final / Closed  
Author: Inacio F. Vasquez  
ORCID: 0009-0008-8459-3400  

Main result:  
For every finite CAT(0) cube complex X,
the homological Dehn function satisfies

δ_H(n) ∈ { O(n), Ω(n^2) }.

No intermediate growth exists.
